# Brand Guidelines

Use these defaults for this Remotion template and future visual work in this repo.

- Heading font: `Gued`
- Sub-heading/body font: `Poppin`
- Primary color: `#606bfa`
- Secondary color: `#a0a6fc`
- Dark color: `#131532`

Avoid rainbow gradients. Use restrained brand gradients made from primary and secondary over the dark background.
